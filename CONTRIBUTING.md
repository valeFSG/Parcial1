# Contribuyendo

Gracias por querer contribuir a este proyecto.

## Cómo contribuir

- Crear una rama desde develop
- Usar nombres como feature/nombre o hotfix/nombre
- Realizar commits claros (feat, fix, docs)
- Crear un Pull Request para integrar cambios

## Reglas

- Mantener el código ordenado
- Seguir buenas prácticas de Git
- Documentar los cambios realizados
